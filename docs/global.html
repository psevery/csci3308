drawPiece(image_id, x, y)

Draws a checker piece to the global canvas context at (x,y).
Parameters:
Name Type Description
image_id integer The integer associated with the piece to be drawn. 1: black, 2: red, 3: black king, 4: red king.
x float The x coordinate of the top left of the square to draw to.
y float The y coordinate of the top left of the square to draw to.
